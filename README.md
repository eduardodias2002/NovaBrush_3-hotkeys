# NovaBrush_3-hotkeys Module
Hotkeys, Color Picker, and floating UI windows!
This is Module 3 of NovaBrush, a lightweight, responsive pixel editor built with C#.

<img src="https://raw.githubusercontent.com/eduardodias2002/eduardodias2002.github.io/refs/heads/dump/img/testpix.webp">

## What It Does

**Hotkeys**  
- `Ctrl+O` — Open file
- `Ctrl+S` — Save file
- `P` — Switch to Pencil
- `B` — Switch to Brush
- `E` — Switch to Eraser
- `F4` — Toggle Anti-Aliasing
- `F5` — Toggle Color Picker window
- `F6` — Toggle Tool window

**Color Picker Window** 
- RGBA sliders for Red, Green, Blue, and Hardness (Alpha)
- Live color preview swatch updates as you drag
- Draggable floating window, grab the top bar and move it anywhere on the canvas
- Clamped to window bounds so it can't be dragged off screen
